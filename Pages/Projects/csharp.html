Covert Bailey's Tape-Measure Fat Formula

Demo Here!


Project Description

This project utilized a Bodyfat Calculator that was provided to us. There are four different calculations, based on the gender of the participant and whether they are above or below 30 years old. The UI provides radio buttons for both gender and age, which will change the display criteria depending on which is selected. For example, if 'Male' is selected, the user must input their waist, hips, forearm, and wrist measurements. Conversely, if 'Female' is selected, the user must input their hips, thigh, calf, and wrist measurements.

At the end, the user clicks 'Calculate', which will display how many pounds of lean body weight and fat the user has, along with the percentage. The app then gives a suggestion about what their maximum weight and body fat percentage should be.

Circuit Analyzer

Demo Here!


Project Description

This application has multiple faces, with the base including a track bar, three resistors, source voltage, and a Calculate Button. Once calculate is clicked, the circuit will now display the voltage running through each resistor, along with the overall current in the top left.

If the track bar is still on screen, the user can click a check box to remove the middle resistor from the circuit. The calculations are roughly the same; the user can input values for source voltage, resistor 1, and resistor 2, with outputs of current, voltage 1, and voltage 2 when "Calculate" is clicked.

If the user selects the 'Remove the Track Bar' check box, there is now a new current calculation for the Amps passing through R2. The user now must provide a Power value, which has a default value of 1.5 Watts. Once 'Calculate' is pressed, the Source Voltage, both currents, and each individual voltage value will be calculated and displayed.

Distance Sensor with Arduino

Demo Here!


Project Description

For this lab, I had to buy an Arduino kit, along with an infrared distance sensor. I wired the distance sensor to the Arduino chip, where the data will be sent through the Serial Port.

I then wrote a MATLAB script to receive the data instead of the Server Monitor. With the distance data, I calculated the velocity and acceleration of the object as it moved, and plotted it on subplots with the distance.

Finally, I derived a first order fit of the distance data to depict the trend of the data and created a best-fit polynomial of the data.
