C# Projects Page


Description

This is the in depth page for all of my C# projects! These all came from my EGR 1400 - Computer Problem Solving course, where the first nine are labs that I completed and the last is part of a Problem Solving assignment. All of these projects were made myself using Visual Studio 2022 and the .NET Windows Forms App Template.



Directions to Oakland University's Campus


Project Description

This was our introductory lab where we created a simple picture and button. When clicked, the button shows rough directions (speicfic to where I lived) for visiting Oakland University's Engineering Center.


Average and Standard Deviation of 10 Random Numbers


Project Description

This project has four buttons with different functionalities; one to fill the ten text boxes with random integer test scores between 50 and 100 and 30 and 100, one to calculate the mean and standard deviation of those scores, one to clear all of the data, and one to exit the application.


Covert Bailey's Tape-Measure Fat Formula


Project Description

This project utilized a Bodyfat Calculator that was given to us. There are four different calculation criteria, based on the gender of the participant and whether they are above or below 30 years old. For example, if 'Male' is selected, the user must input their waist, hips, forearm, and wrist measurements. Conversely, if 'Female' is selected, the user must input their hips, thigh, calf, and wrist measurements.

At the end, the user clicks 'Calculate', which will display how many pounds of lean body weight and fat the user has, along with the percentage. The app then gives a suggestion about what their maximum weight and body fat percentage should be.


Craps Card Game


Project Description

This project involved us re-creating the popular card game Craps. For this, we were given the basic rules, along with a flowchart and a background image for our GUI.

The game keeps track of the user's total score, along with the current Point, which is displayed using pucks at the top. Each time the die are rolled, two random numbers are generated, along with their corresponding dice images. Depending on the roll and situation (initial roll or second and onwards), the user's score and Point will be updated.


Circuit Analyzer


Project Description

This application allows the user to work with either a two or three resistor circuit. The user can choose whether they want to have a track bar or not, which is used to adjust the Source Voltage. If the user clicks to remove the track bar, the Source Voltage is now what's being calculated, and an input for power appears.

When 'Calculate' is clicked, all blank variables will have their correct values displayed. There are variables for Source Voltage (with no Track Bar), overall current, current through R2 (with no Track Bar), and voltage for either two or three resistors.


2nd Degree Parabola Plotter


Project Description

This application allows the user to graph a 2nd-degree parabola. There are input values for a, b, and c in 'y = ax^2 + bx + c', along with how much of the graph you'd like to see: xmin, xmax, ymin, ymax, and how many points you want plotted. Once the user clicks on the 'Graph' tab, the parabola is automatically plotted using Microsoft Chart Control.


Anagram Tester


Project Description

This project involved us importing a Histo.dll file, which was made in Visual Basic. This Histo library contains a method, CreateHistogram, which creates a histogram, in the form of an array, of each alphabetical character in a word. We were also given two txt files, one for potential anagrams, and one with the confirmed anagrams.

The design is simplistic, with a ListBox and a button. When the button is pressed, all of the found anagrams, along with what word/phrase they came from, are displayed.


Hospital Database Parser


Project Description

In this application, we were given a txt and xml file containing information about blood tests of Hepatitis patients, which we parsed and kept track of their information. We also added a Database Source to the app, a Microsoft Access file that held hospital medical records, specifically whether the patient lived or died.

The UI consists only of a single "Analyze" button. When pressed, the code combines the parsed information and returns the average age and standard deviation of patients that have both lived and died for each blood test characteristics: Bilirubin, SGot, Alk Phosphate, and Albumin.


Distance Sensor with Arduino


Project Description

For this lab, I had to buy an Arduino kit along with an infrared distance sensor. I wired the distance sensor to the Arduino chip, where the data will be sent through the Serial Port.

I then wrote a MATLAB script to receive the data instead of the Server Monitor. With the distance data, I calculated the velocity and acceleration and plotted them all on subplots.

Finally, I derived a first order fit of the distance data to depict the trend of the data and created a best-fit polynomial of the data.

Microsoft Excel Matrix Script


Project Description

For this project we were given 8 pairs of x and y values. With these, we had to find a0, a1, and a2 for a parabolic least squares regression using Microsoft Excel. Instead of manually typing in and doing the calculations within Excel, we wrote Visual Basic code to automate the process.

The UI is very basic, with a large button that, when pressed, will start the process. Within the Excel file, it writes down the two matrices we manually entered, the values of them multiplied together, and then the inverse of that result. This will end up solving for the coefficients for parabolic least squares regression (of degree 2). The generated Microsoft Excel file will also be saved in a specific folder within the user's system.
