Project Requirements

This project was a part of the latter half of my Web and Mobile Systems class. Similarly, we were given open reign, with the goal to build a dynamic app that can run on Android systems.


Project Description

To develop this app, I used the React Native framework, accompanied with Expo to aid in making the application accessible to Android phones.

Building off of my web application, I wanted to maintain a similar theme. I ended up deciding on a simplistic to-do list project. However, there is beauty within simplicity; anyone can pick up this app and instantly understand how to use it.
