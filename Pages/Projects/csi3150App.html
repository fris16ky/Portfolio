Project Description

To develop this app, I used the React Native framework, accompanied with Expo to aid in making the application accessible on Android phones.

Building off of my web application, I wanted to maintain a similar theme. I ended up settling with a simplistic to-do list project. However, there is beauty within simplicity; anyone can pick up this app and instantly understand how to use it.
