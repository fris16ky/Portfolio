Garry's Mod Server


Project Demo

I specifically talked through the definition of GMod and TTT, along with the webpage.



Project Requirements

We were tasked with creating a group of 3-4 people, where our task is to provide a service while hosting our project on a Google Cloud Platform Virtual Machine. In addition to the service, we had to write at least one automated script, periodically log system features, write a script to back up our server, and create a webpage.


Project Description

Our classwork was done in CentOS7, but we switched to Ubuntu for the duration of the project. As was a popular decision, our group decided on providing a multiplayer game server for a subgame within Garry's Mod; Trouble in Terrorist Town (TTT).

Our server is run using the Google Cloud Platform, where we received credits for our CSI 3660 - System Administration class. Running this project does cost money, so the server is only functional when we choose to run our VM.

Our server provides a game server for TTT, in which players can play the game through our IP address. Our server provides two unique maps, with themes of Minecraft and the popular First-Person Shooter, Counter-Strike: Global Offensive.
