Important:

In order to run our server using the GCP, our professor gave us 50$ worth of credits at the beginning of the year, which allows us to run our project for free and for a while.

Also, whilst during the class we used CentOS7, we switched to Ubuntu for our game server code n such.

In this class, our final exam/project was to run an accessible game server using the Google Cloud Platform. When the class started, our professor gave us credits that could last us until the end of the year, if not longer. It was our job to use our Linux knowledge and what we've learned about system administration to create a sustainable game server.

My group ending up choosing to make a server for a Steam game called Garry's Mod. Steam is a digital library where people from around the world can all search for and download, create mods, and have a community for a wide variety of video games.
