Project Description

This mobile application was created for my CSI 4999 class; Senior Capstone Project. For this class we created a group of four people and were tasked with developing any application that has a front and back end. This could be a mobile or web application, or a game. Our group ended up with a Mobile Application that we named Investo.


Technologies

To create this project, our IDE of choice was Android Studio. We coded using Java, and had a back-end database of SQLite3.


This is the temporary description for my Senior Capstone Project - Investo. Investo is a mobile application that can be used for budgeting, financing and more!
